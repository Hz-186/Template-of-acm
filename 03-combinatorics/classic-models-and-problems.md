### 逆序对

* 逆序对的其中一个重要性质是，逆序对的值等于其在冒泡排序的过程中，相邻元素的交换次数，且排序过程中每次交换相邻元素后逆序对的数目减少1。

* Bubble Sort 求逆序对数

```C++
auto Bubblesort = [&]() -> void {
	int n = a.size() - 1;
	for (int i = 1; i <= n; i++) {
		for (int j = 1; j < n; j++) {
			if(a[j] < a[j + 1]) {
				swap(a[j], a[j + 1]);
				// 每次交换，逆序对++ / --
			}
		}
	}
};
```

### 无序配对问题

给定一个由 $2m$ 个 **有标签** 元素（比如编号为 $1,2,\dots,2m$）组成的集合，**无序配对**（也叫“把集合划分为若干个无序的 2 元子集”或“perfect matching”）是把这 $2m$ 个元素两两配成 $m$ 个子集，每个子集包含两个元素。$eg: \{1, 2, 3, 4, 5, 6\}$ 可以划分为 $\{ \{1, \ 2\}, \{3, \ 4\}, \{5, \ 6\} \}$，这就是一种配对方案，现在要求 **方案数** 有多少。

把 $2m$ 个元素配对，最直观的想法是：
- 先选元素 $1$ 的伴侣：有 $2m-1$ 种选择（可以配 $2..2m$ 中任意一个）。

- 配完 $1$ 之后，剩下 $2m-2$ 个元素；取其中最小编号（比如 取 $2$ 的下一个未配的编号）做同样的操作，它的伴侣有 $2m−3$ 种选择。

- 然后再有 $2m-4$ 个剩下的，下一步有 $2m-5$ 种，…，最后是 $1$。

把这些选择相乘得到：
$$(2m-1)\cdot(2m-3)\cdot\cdots\cdot 3\cdot 1 = (2m-1)!!.$$
这个乘积就是**无序配对数**（perfect matchings 的数量）。

把所有 $2m$ 个元素**全排列**一遍，排列数为 $(2m)!$。现在把排列按顺序两两一组，即
$$(\,a_1,a_2\,),(\,a_3,a_4\,),\dots,(\,a_{2m-1},a_{2m}\,)$$
这会把排列转换成一个**有序的、并且每对内有序**的配对列表。每个无序配对（我们关心的目标）会被这种排列方式重复计数多次，重复的次数是：

- 每一对内部两个元素的顺序可以互换（$2$ 种），所以总共有 $2^m$ 种内部顺序的排列；

- 各个对之间的顺序也无关紧要（我们不区分先后），排列成 $m$ 对的顺序有 $m!$ 种。

因此无序配对的总数等于
$$\frac{(2m)!}{2^m\cdot m!}.$$

注意这是和上面双阶乘相等的，因为
$$(2m-1)!! = \frac{(2m)!}{2^m m!}.$$

举个具体的实际场景例子：

https://codeforces.com/contest/1957/problem/C

放一对棋子就会占据两行两列，也就是 配对问题了。

## 斯特林数相关的组合数学问题

这几天借学长的账号补去年2024牛客暑期多校第一场 B 题时，接触到了一个很有意思的组合数学问题，简单抽离出来就是

"将 *i* 个**不同的**小球放入到 *j* 个**不同的**盒子中会产生多少种方案？"

如果有人熟悉第二类斯特林数就会感觉到这个问题很熟悉，但是又不完全一样，因为第二类斯特林数所解决的问题是：

"将 *i* 个**不同的**小球放入到 *j* 个**相同的**盒子中会产生多少种方案？"

盒子由 **不同的** —> **相同的**

接下来将对 盒子**相同 / 不同**， 球**相同 / 不同**， 盒子允不允许为**空** 等情况进行总结与计算 。

### 相同球 ➔ 相同盒子，不允许空盒子

这个模型其实可以转化的更加常见，可以视作，将一个大整数 n 拆分为若干个 k 个数字。
也就是**整数拆分问题**，求方案数。

递推表达式可以写作 $$f[i][j] = f[i - 1][j - 1] + f[i - j][j]$$
其中 $f[i][j]$ 表示为把 i 这个整数拆分为 j 部分的方案数。

可以怎么理解呢？

* 如果所有盒子中小球数量仅满足 >= 1，这时候 $f[i - 1][j - 1]$ 种方案中每种方案后面都塞一个仅装有一个小球的盒子，就可以满足 $f[i][j]$

* 接下来就是所有盒子中小球数量都 >= 2 的情况了，这种情况下，我们可以先将每个盒子中的小球平均 -1，这样一来盒子的中小球数量又恢复了 >= 1，可以理解为递归求解。

```cpp
f[i][j] = f[i - 1][j - 1]    // 第一个盒子只放 1 个球，剩下把 i-1 球拆为 j-1 份
        + f[i - j][j];       // 每个盒子先放 1 个球，剩下 i-j 球再拆成 j 份
// 边界
f[0][0] = 1;
f[i][0] = 0;     // i > 0
f[0][j] = 0;     // j > 0
```

### 相同球 ➔ 相同盒子，允许空盒子

递推表达式可以写作 $$f[i][j] = f[i][j - 1] + f[i - j][j]$$
这里可以理解为后面可以放一个空的盒子。

### 相同球 ➔ 不同盒子（隔板法）

这就是个纯组合数学问题了。
#### 如果是允许有空盒子出现的话
$$\begin{pmatrix}
i + j - 1 \\
j - 1
\end{pmatrix}$$
#### 不允许有空盒子

所以先在每个盒子中预处理一个球，所以目前没有放的球的数量就是 $(i−j)$

则最后答案就是
$$\begin{pmatrix}
i − 1 \\
j - 1
\end{pmatrix}, (i >= j)$$

| **情况**    | **物理模型**             | **数学要求**            | **公式**            |
| --------- | -------------------- | ------------------- | ----------------- |
| **不允许为空** | $M$ 个球，找空隙插 $k-1$ 块板 | 都是正整数 ($x_i \ge 1$) | $C_{M-1}^{k-1}$   |
| **允许为空**  | $M$ 个球和 $k-1$ 块板一起排队 | 是非负整数 ($x_i \ge 0$) | $C_{M+k-1}^{k-1}$ |

### 不同球 ➔ 相同盒子，不允许空盒子

这也就是最最经典的 **第二类斯特林数问题**

递推表达式也就是$$f[i][j] = j * f[i - 1][j] + f[i - 1][j - 1]$$
$f[i][j]$ 表示把 i 个小球放入 j 个相同的盒子时的方案数到底是多少

```cpp
// f[n][k] = S(n,k)
f[i][j] = j * f[i - 1][j]    // 第 i 个球放入已有 j 个无标签盒子中任意一个
        + f[i - 1][j - 1];   // 单独开一个新盒子只装第 i 个球
// 边界
f[0][0] = 1;
f[i][0] = 0;    // i > 0
f[0][j] = 0;    // j > 0
```


### 不同球 ➔ 不同盒子，不允许空盒子

这个也就是2024牛客暑期多校第一场B题中的问题了。
我们来分析一下：

首先跟第二类斯特林数的分析方式一致

第 i 个小球可以放入 j 个已经有小球的盒子中，则此时一共 j 个盒子，也就是 j 种放法，所以 要 $j * f[i - 1][j]$

同时我们又可以发现 当新开一个新的盒子放入的时候，每种方案的放法都有 j 种，可以理解为目前这个 仅有一个球的盒子在 j 个不同的位置上（位置不同，则盒子不同）。

先在 j 个盒子里挑一个空盒子（j 种选法），把第 i 号球放进去；然后要把前 i - 1 个球分配到剩下的 j - 1 个盒子里（都要非空），这样的方式有 f[i−1][ j−1] 种。
因此这一部分总计

```cpp
f[i][j] = j * f[i - 1][j]
        + j * f[i - 1][j - 1];
// 边界
f[0][0] = 1;
f[i][0] = 0;    // i>0
f[0][j] = 0;    // j>0
```

乘 `j` 是因为 `f[i][j]` 这里面有个 `j`，然后讨论，能不能放到一个空盒子中，如果不能放到一个空盒子中.
