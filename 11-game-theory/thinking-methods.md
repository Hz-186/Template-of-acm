# 思考方式

- 并联 → 用 SG。
- 串联 → 退回到输赢类（N 态/P 态）递推分析。

 划分集合法（原则）
1. **按“能否被谁下”来分**：先问每个局部（段）对 Alice/Bob 谁可下，谁不可下。若只有一方能下，通常是危险/关键段。

2. **按“下了之后对手能否再下”来分**：把能下的段再细分为“下一次就消失/不能再被下”的段和“下一次后仍会留下可下区域”的段。

3. **把能产生对手“必胜/必败”机会的段单列出来（关键段）**：例如会生成 `b≤len<a` 的段、或会留下另一个 `len≥2b` 的段的那类，称为“危险段/长段”。

4. **用阈值判断而不是具体位置**：阈值常见是 b、a、2b、a+b（视题而定），阈值来源是“对手步长 / 本人步长 / 两步合并能否形成可下区间”。

5. **优先处理数量极少的“特殊段”**：0 / 1 / ≥2 这三种数量级往往能直接决定胜负（因为一次行动不可能同时解决 ≥2 个远隔的关键段）。

 实战中该注意的点（Checklist）

- **奇偶性经常出现**：如果除了少数特殊段外，局面只剩“有限次可下”的段，通常由某类段的奇偶性决定胜负。

- **记住 partizan vs impartial**：若两人动作不同（a≠b），SG（Grundy）通常不适用，转而用 case 分析与枚举；若动作对称可尝试求 Grundy 值并异或合成。

- **测试策略**：把 n 很小的情况穷举暴力（对拍）找模式，再归纳成一般规则；对修复后的逻辑做小规模全枚举检验。
