# 一种完全二叉树 比赛 淘汰的 Trick

1. **单淘汰 / 层级赛制**：比赛/过程可以看成从叶到根的若干轮，每轮两两配对，胜者进入下一轮（完全二叉树或等价分层结构）。
2. **可重新配对 / 可控制配对**：你可以在每一轮自由安排配对（或能通过某种方式把潜在对手集合分离），这保证了“谁会在某一层出现”为可控/可分层的。
3. **决胜关系单调/确定**：胜负由某种全序（强度、优先级、权值）决定——没有随机性或循环胜关系（否则无法简单分层）。
4. **有“代价”或“资源”可以消除风险**：可以付代价（或选用资源）来使某个潜在对手对目标无威胁，且代价是静态确定的。
5. **每层需要“至少一个”保护者**：目标经过 `log n` 层，每层需要保证有某个人或资源来保证那层安全（常见是每层至少 1 个）。
6. **候选人的“危险级”能按强度分层**：所有会在前 `t` 层出现的危险者能被你划为前若干块（大小恰为 `n/2, n/4, ...`），且这些块可以逐块开放到候选集合中。
7. **可交换性 / 局部调整不影响可行性**：若在某层你用更便宜的候选替换更贵的候选，不会让后面的层级失去可行的候选（这是贪心正确性的关键条件）
	 **“分块（按 2 的幂）+ 小顶堆贪心”**

```C++
	priority_queue<int, vector<int>, greater<int>> heap;  
	int l = n, m = n / 2, ans = 0;
	for (int i = n; a[i] != -1; i--) {
		heap.push(a[i]);
		if (i == l) {
			ans += heap.top();
			heap.pop();
			l -= m; m /= 2;
		}
	}
```
