# 数学


**二项式定理**

$$(a + b)^n = \sum_{k = 0}^{n}C_n^ka^{n - k}b^{k}$$


**范德蒙德卷积公式**（Vandermonde's Convolution）

$$∑_{k = 0}^{r}​ \left (\begin{matrix}m\\k\\ \end{matrix}\right) \left (\begin{matrix}n\\r - k\\ \end{matrix}\right) = \left (\begin{matrix}m + n\\r\\ \end{matrix}\right)$$
